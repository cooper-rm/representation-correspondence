# notes/

My own notes on papers, ideas, and scoping. Markdown files, organized loosely.

Suggested naming: `YYYY-MM-DD-short-topic.md` so the directory sorts chronologically.

Examples of what lives here:
- Per-paper deep-read notes (one file per paper I want to engage with in depth)
- Scoping decisions and the reasoning behind them
- Idea drafts that aren't yet ready for `RESEARCH_QUESTION.md`
- Conversations with advisor (Busch) — what we discussed, what was decided
